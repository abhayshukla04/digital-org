## 1. How to start in local
```
$ git clone [https://github.com/abhayshukla04/digital-org.git](https://github.com/abhayshukla04/digital-org.git)
$ cd digital-org
$ mvn spring-boot:run

```

http://localhost:8080/swagger-ui/#/
